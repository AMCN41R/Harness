Harness Namespace

The Harness namespace contains classes and types theat allow the configuration and population of one or more mongo databases.
Classes
  ClassDescription
HarnessBase
A base class that can be used in conjunction with the HarnessConfigAttribute to configure one or more Mongo databases.
HarnessManager

The HarnessManager class, alongside the SettingsBuilder class, is used to configure and build one or more mongo databases, putting them into a known state.

Once you have your HarnessConfiguration settings object, you can pass it to the HarnessManager and use build your databases. The Build method returns a DictionaryTKey, TValue containing the MongoClient instances (one per unique connection string in the HarnessConfiguration) for re-use. The dictionary key is the connection string.
