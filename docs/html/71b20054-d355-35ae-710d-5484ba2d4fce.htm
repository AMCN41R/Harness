Harness.Settings Namespace

The Harness.Settings namespace contains classes and types that allow the creation of configuration for Harness.
Classes
  ClassDescription
CollectionConfig
Represents the collection configuration options.
ConventionConfig
Represents the convention configuration options.
DatabaseConfig
Represents the database configuration options.
HarnessConfiguration
Represents the Harness configuration options.
SettingsBuilder
A fluent api that lets you build the Harness configuration object in code instead of using an external json file.
SettingsBuilderException
Represents errors thrown by the SettingsBuilder class and its extensions.
SettingsLoaderException
Represents exceptions thrown by the ISettingsLoader.
